Projects

Distributed Key-Value Store Application

January, 2016 - April, 2016

In a small group, we designed and developed a distributed application that stored key-value pairs across 100 nodes located across the world. As some nodes were unstable, the application was designed so nodes could enter and leave the cluster without service interruption. Key-value pairs were also replicated on three nodes so that if a node left, its store wasn't lost.
